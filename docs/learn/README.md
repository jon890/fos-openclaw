# Cross-workspace Learnings

여러 워크스페이스에 적용 가능한 학습/회고 모음. 워크스페이스 한정 학습은 `<workspace>/docs/learn/`에 둔다 (예: `career-os/docs/learn/`).

ADR(`<workspace>/docs/decisions/`)이 "결정과 그 근거"를 기록한다면, 이 디렉터리는 "사고/실수/패턴에서 얻은 교훈"을 기록한다. ADR은 변경 시 Superseded로 마킹하지만, 학습은 누적된다.

## 문서 형식

각 학습 문서는 다음 섹션을 포함한다:

- 날짜
- 상황 (무엇이 일어났는가)
- 무엇을 잘못했는가 / 시간 손실 지점
- 학습 / 패턴
- 향후 적용 (구체적 트리거 + 행동)
- 참고 (관련 ADR / 커밋 / 세션)
