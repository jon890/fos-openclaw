# Public-Safe Health-Care Policy

이 파일은 외부에 공개되어도 괜찮은 운영 원칙만 담는다.

## 공개 가능

- 워크스페이스 목적과 디렉터리 구조
- 개인정보를 제거한 일반 건강관리 운영 원칙
- 진료 준비/경과 기록/재활 체크리스트를 관리한다는 수준의 설명
- 특정 개인 식별자, 병원명, Discord ID, 정확한 진료기록 원문이 없는 일반화된 회복 플랜

## 공개 금지

- 원본 진료기록, 검사 이미지, 처방전, 주민/병원 등록번호, 실명 등 개인 식별 정보
- Discord 채널/서버/사용자 ID 같은 플랫폼 내부 식별자
- 병원명/의료진명/정확한 진료일이 포함된 원본성 기록
- 사용자의 상세 증상 경과가 그대로 드러나는 문서

## 운영 원칙

- `config/`에는 공개되어도 괜찮은 정책/템플릿/일반화된 플랜만 둔다.
- 민감한 원본과 상세 컨텍스트는 `data/`에만 둔다. `data/`는 gitignore 대상이어야 한다.
- 공개/비공개 경계가 애매하면 임의로 커밋하지 말고 사용자와 먼저 논의한다.
