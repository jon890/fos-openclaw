# 여행 의사결정 로그

## 2026-04-28

### 결정 주제
- 오사카 여행 계획을 여행별 폴더 구조로 관리
- 1차 초안 기준으로 일정 밀도를 조정해 추천안 작성

### 반영한 결정
1. Day 1
   - 우메다 쇼핑 스팟을 압축
   - 한큐/한신은 1곳 우선
   - 우메다 스카이빌딩은 기본안에서 제외
   - 난바/도톤보리 저녁 마무리 흐름 채택

2. Day 2
   - USJ는 하루 단독 일정으로 유지
   - 저녁은 체력에 따라 우메다 가벼운 식사/쇼핑 선택

3. Day 3
   - 교토는 청수사/산넨자카/니넨자카/기온/야사카신사/% Arabica 중심으로 정리
   - 아라시야마 치쿠린/도게츠교는 선택 옵션으로 격하

4. Day 4
   - 오사카성 단독에서 마지막 쇼핑 구간을 추가하는 방향으로 보강

### 기존안 대비 차이
- Day 1: 우메다 -> 난바 -> 우메다 왕복 구조를 줄여 피로도 완화
- Day 2: 큰 변경 없음, USJ 집중 유지
- Day 3: 동쪽 교토 + 서쪽 교토를 하루에 다 도는 구성에서 동쪽 교토 중심으로 단순화
- Day 4: 오사카성만 보던 일정에 마지막 쇼핑/기념품 시간을 추가

### 추가 판단
- 교통은 **ICOCA 등 교통카드 + 개별 결제**를 기본 추천
- Osaka Amazing Pass / e-Pass는 현재 일정만 보면 필수는 아님
- USJ는 **Studio Pass는 필수**, 인기 구역/어트랙션 목표가 뚜렷하면 **Express Pass 검토 가치 높음**
- 슈퍼 닌텐도 월드를 노리면 **Timed Entry / Standby Entry / Express Pass 포함 여부**를 꼭 확인

### 현재 의사결정 진행 방식
- 갈래가 나뉘는 항목을 후보군 + 예상 금액과 함께 먼저 제시
- 사용자 응답을 받아 범위를 좁힌 뒤 `decision-log.md`에 확정안 기록
- 필요 시 `itinerary.md`에 즉시 반영

### 일정 반영 방향
- Day 1은 우메다 쇼핑 시간을 늘리고, 도톤보리는 식당 1곳 중심으로 안정화했다.
- Day 3은 동쪽 교토와 아라시야마를 모두 포함하는 빡센 확장 코스로 변경했다.
- Day 4는 마지막 쇼핑 위치가 미정이므로 공항 이동 안정성이 높은 우메다 마무리를 기본안으로 유지했다.
- USJ Express Pass는 최종 결정 전까지 Studio Pass + 오픈런 기본 전략과 Express Pass 검토 전략을 병행 표기한다.
