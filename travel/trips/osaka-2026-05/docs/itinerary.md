# 오사카 여행 일정표 - 의사결정 반영본

## 반영한 오늘 아침 선택
- 1B: KIX -> 우메다 이동은 JR/공항철도 계열 우선
- 2B: Day 1 우메다 쇼핑은 넉넉히
- 3B: Day 1 저녁은 도톤보리 구경 + 근처 식당 1곳
- 4: USJ Express Pass는 보류 / 고민중
- 5B: Day 3 교토는 동쪽 교토 + 아라시야마까지 도전
- 6: Day 4 마지막 쇼핑 위치는 미정. 현재 안전한 기본값은 우메다 마무리

## Day 1 - 2026-05-13 (수) / 입국 + 우메다 쇼핑 + 도톤보리

### 일정
- 09:15~11:10 ICN -> KIX 비행
- 11:10~12:20 입국 / 수하물 / 이동 준비
- 12:20~13:30 JR/공항철도 계열로 우메다 이동
- 13:30~14:00 호텔 한큐 리스파이어 오사카 짐 맡기기
- 14:00~15:00 점심: 그랜드 프론트 / 루쿠아 / 무지 카페 후보
- 15:00~17:40 우메다 쇼핑 넉넉히
  - 그랜드 프론트 오사카
  - 루쿠아 오사카
  - 한큐 백화점 우메다 본점
  - 한신 백화점 우메다 본점 또는 주변 추가 쇼핑
- 17:40~18:10 우메다 -> 난바 이동
- 18:10~20:40 도톤보리 / 구리코 / 에비스바시 산책
- 19:00~20:30 도톤보리 근처 식당 1곳 진입
  - 라멘 / 오코노미야키 / 이자카야 / 쿠시카츠 중 선택
- 20:40~21:20 호텔 복귀

### 포인트
- 첫날은 2B 선택 때문에 쇼핑 시간이 길어짐.
- 대신 난바는 “길거리 이것저것”보다 식당 1곳을 잡는 방식으로 안정화.
- 너무 피곤하면 한신/한큐 중 하나는 다음날 저녁 또는 Day 4로 넘기는 게 좋음.

## Day 2 - 2026-05-14 (목) / USJ 단독일

### 일정
- 07:00~07:30 기상 / 준비
- 07:30~08:15 오사카역 -> 유니버설시티역 이동
- 08:15~09:00 입장 대기 / 오픈런
- 09:00~12:30 인기 어트랙션 우선
- 12:30~13:30 점심
- 13:30~18:00 닌텐도월드 / 해리포터 / 퍼레이드 / 남은 어트랙션
- 18:00~19:00 USJ 내부 저녁 또는 간식
- 19:00~20:00 호텔 복귀
- 20:00 이후 체력 되면 우메다에서 가벼운 식사 / 편의점 / 백화점 식품관

### 아직 결정 필요
- Express Pass 여부는 아직 고민중.
- 닌텐도월드나 인기 라이드 욕심이 크면 Express Pass 쪽이 안전.
- 비용 아끼는 방향이면 Studio Pass + 오픈런 + 앱 timed entry 전략.

- 10:50~11:40 산넨자카 / 니넨자카 산책
