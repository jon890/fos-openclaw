# 오사카 여행 맛집 / 쇼핑 후보

> 기준: 현재 확정 일정표에 맞춘 후보군. 영업시간/휴무/예약 가능 여부는 출발 전 재확인 필요.

## Day 1 - 우메다 쇼핑 + 도톤보리 저녁

### 우메다 점심 후보
1. **오코노미야키 키지 우메다 스카이빌딩/우메다권**
   - 추천 상황: 오사카 첫 끼를 오코노미야키로 시작하고 싶을 때
   - 장점: 오사카스러운 메뉴, 만족도 높음
   - 주의: 웨이팅 가능
2. **무지 카페 그랜드 프론트 오사카점**
   - 추천 상황: 가볍고 깔끔하게 먹고 바로 쇼핑 이어갈 때
   - 장점: 동선 좋고 부담 적음
   - 주의: 여행지 맛집 느낌은 약함
3. **루쿠아 / 그랜드 프론트 식당가**
   - 추천 상황: 메뉴를 현장에서 보고 고르고 싶을 때
   - 장점: 선택지 많고 실패 확률 낮음
   - 후보: 돈카츠, 파스타, 일식 정식, 카페류
4. **한큐/한신 백화점 식품관 간단 식사**
   - 추천 상황: 쇼핑 중심으로 움직일 때
   - 장점: 디저트/도시락/간식 같이 해결 가능

### 우메다 쇼핑 후보
1. **그랜드 프론트 오사카**
   - 볼거리: 편집숍, 라이프스타일, 카페, 식당
   - 추천: 첫날 동선 시작점
2. **루쿠아 오사카 / 루쿠아 1100**
   - 볼거리: 의류, 잡화, 젊은 브랜드, 식당가
   - 추천: 실제 쇼핑 만족도 높음
3. **한큐 백화점 우메다 본점**
   - 볼거리: 여성 패션, 화장품, 디저트, 고급 식품관
   - 추천: 선물/디저트/백화점 쇼핑
4. **한신 백화점 우메다 본점**
   - 볼거리: 식품관, 간식, 술/안주류
   - 추천: 먹거리 쇼핑
5. **다이마루 우메다 / 포켓몬센터 오사카 / 닌텐도 오사카**
   - 추천 상황: 캐릭터 굿즈 관심 있으면 강추
6. **요도바시 카메라 + LINKS UMEDA**
   - 볼거리: 전자제품, 잡화, 생활용품, 식당
   - 추천: 구경거리 많음

### 도톤보리 저녁 후보
1. **오코노미야키 미즈노**
   - 메뉴: 오코노미야키
   - 추천 상황: 유명한 오코노미야키 맛집을 가고 싶을 때
   - 주의: 웨이팅 가능성 높음
2. **아지노야**
   - 메뉴: 오코노미야키 / 야키소바
   - 추천 상황: 도톤보리 근처에서 식사 만족도 안정적으로 잡고 싶을 때
   - 주의: 웨이팅 가능
3. **쿠시카츠 다루마 도톤보리점**
   - 메뉴: 쿠시카츠
   - 추천 상황: 오사카식 술안주/튀김 느낌
   - 주의: 기름진 편
4. **이치란 도톤보리점**
   - 메뉴: 라멘
   - 추천 상황: 무난하고 늦은 시간에도 대응하기 좋음
   - 주의: 관광객 많음
5. **킨류라멘 도톤보리**
   - 메뉴: 라멘
   - 추천 상황: 빠르게 먹고 도톤보리 분위기 즐길 때
6. **타코야키 와나카 / 쿠쿠루 / 크레오루**
   - 메뉴: 타코야키
   - 추천 상황: 식당 전후 간식

### Day 1 추천 조합
- 안정형: 무지 카페 점심 -> 루쿠아/한큐 쇼핑 -> 아지노야 또는 다루마
- 오사카 느낌형: 키지 점심 -> 한신 식품관 -> 미즈노 + 타코야키
- 체력 아끼기형: 그랜드 프론트 식당가 -> 루쿠아 중심 -> 이치란/킨류

## Day 2 - USJ

### USJ 내부 식사 후보
1. **해리포터 구역 식사**
   - 추천 상황: 테마 분위기를 살리고 싶을 때
   - 장점: 여행 느낌 좋음
   - 주의: 가격대 있음
2. **닌텐도월드 간식/식사 후보**
   - 추천 상황: 닌텐도월드 입장 성공 시
   - 장점: 사진/기념성 좋음
   - 주의: 대기 가능
3. **파크 내 빠른 식사 + 간식 조합**
   - 추천 상황: 어트랙션 우선일 때
   - 장점: 시간 절약

### USJ 준비/쇼핑 후보
1. **공식 앱 설치 / 티켓 QR 저장**
2. **닌텐도월드 timed entry 확인**
3. **굿즈 쇼핑**
   - 닌텐도, 해리포터, 미니언, 스누피 계열
4. **구매 타이밍**
   - 오전: 머리띠/소품 먼저 사서 사진 활용
   - 저녁: 무거운 굿즈 마지막에 구매

### USJ 후 우메다 저녁 후보
1. **백화점 식품관 도시락/간식**
   - 추천: 가장 편함
2. **오코노미야키 키지**
   - 추천: 체력 남으면
3. **요도바시/LINKS 식당가**
   - 추천: 메뉴 선택지 넓음
4. **호텔 근처 편의점 + 디저트**
   - 추천: 피곤하면 이게 제일 현실적

## Day 3 - 교토 동쪽 + 아라시야마

### 동쪽 교토 점심 후보
1. **오멘 고다이지점 / 기온권**
   - 메뉴: 우동/면류
   - 추천 상황: 깔끔한 교토식 점심
   - 주의: 대기 가능
2. **히사고**
   - 메뉴: 오야코동
   - 추천 상황: 청수사/기온 사이에서 일본식 덮밥
   - 주의: 인기점이라 대기 가능
3. **기온/히가시야마 소바·정식집 현장 선택**
   - 추천 상황: 시간 밀릴 때 유연하게 대응
4. **간단식 + 카페 전략**
   - 추천 상황: 아라시야마까지 가야 해서 시간을 아끼고 싶을 때

### 동쪽 교토 카페/간식 후보
1. **% Arabica Kyoto Higashiyama**
   - 추천: 일정 여유가 있으면
   - 주의: 줄 길면 과감히 스킵
2. **요지야 카페 기온/교토권**
   - 추천: 교토 감성 카페
3. **산넨자카/니넨자카 말차 디저트 가게들**
   - 추천: 이동 중 간식으로 좋음

### 아라시야마 식사/카페 후보
1. **아라시야마 요시무라**
   - 메뉴: 소바
   - 추천 상황: 아라시야마에서 제대로 한 끼
   - 주의: 인기점, 대기 가능
2. **스테이크 오츠카**
   - 메뉴: 스테이크/고기
   - 추천 상황: 고기 좋아하면 후보
   - 주의: 웨이팅/영업시간 확인 필요
3. **히로카와**
   - 메뉴: 장어
   - 추천 상황: 예산 있고 예약 가능하면
   - 주의: 예약/가격/시간 확인 필수
4. **eX cafe Kyoto Arashiyama**
   - 메뉴: 디저트/말차/카페
   - 추천 상황: 분위기 좋은 휴식
5. **% Arabica Kyoto Arashiyama**
   - 추천 상황: 히가시야마점을 못 갔을 때 대체

### 아라시야마 쇼핑/볼거리
1. **치쿠린**
2. **도게츠교**
3. **아라시야마 상점가**
   - 간식, 말차, 기념품, 잡화
4. **교토 기념품**
   - 말차 과자, 유자/시치미, 전통 잡화, 손수건류

### Day 3 추천 조합
- 빡센 효율형: 청수사 -> 산넨자카 -> 간단 점심 -> 기온 짧게 -> 아라시야마 -> 요시무라/카페
- 감성형: 청수사 -> 산넨자카 -> % Arabica -> 기온 -> 아라시야마 산책 -> eX cafe
- 식도락형: 히사고/오멘 -> 아라시야마 -> 히로카와 또는 스테이크 오츠카

## Day 4 - 오사카성 + 우메다 마지막 쇼핑

### 오사카성 근처 점심 후보
1. **오사카성 공원/조테라스 오사카 식당가**
   - 추천 상황: 동선 최우선
   - 장점: 바로 이동 가능
2. **우메다 복귀 후 점심**
   - 추천 상황: 마지막 쇼핑과 묶기
   - 장점: 선택지 많음
3. **백화점 식품관/도시락**
   - 추천 상황: 공항 이동 전 시간을 아끼고 싶을 때

### 마지막 쇼핑 후보 - 우메다 기본안
1. **한큐 백화점 식품관**
   - 추천: 디저트/고급 선물
2. **한신 백화점 식품관**
   - 추천: 먹거리/간식/로컬 느낌
3. **551 호라이**
   - 추천: 만두/기념 먹거리
   - 주의: 반입/냄새/대기 고려
4. **다이마루 우메다 캐릭터 굿즈**
   - 포켓몬센터 오사카
   - 닌텐도 오사카
5. **요도바시 카메라**
   - 전자제품/잡화/일본 생활템
6. **돈키호테 우메다권**
   - 추천: 약, 과자, 잡화 몰아서 구매
   - 주의: 시간이 잘 녹음

### 공항 쇼핑 후보
1. **공항 면세/기념품**
   - 부족한 과자/선물 보충
2. **간사이공항 식사/간식**
   - 시간 여유 있을 때만

### Day 4 추천 조합
- 안정형: 오사카성 -> 우메다 점심 -> 한큐/한신 식품관 -> 공항
- 쇼핑형: 오사카성 짧게 -> 다이마루/포켓몬/닌텐도 -> 한신 식품관 -> 공항
- 체력 아끼기형: 오사카성 -> 조테라스 점심 -> 호텔 짐 회수 -> 공항에서 쇼핑

## 전체 우선순위 추천

### 꼭 넣을 만한 것
- 우메다: 루쿠아, 한큐/한신 식품관
- 도톤보리: 오코노미야키 또는 쿠시카츠 1곳 + 타코야키 간식
- USJ: 닌텐도/해리포터 굿즈 중 관심 있는 것
- 교토: 청수사 주변 말차/기념품 + 아라시야마 카페 또는 소바
- 마지막 날: 우메다 식품관에서 선물 정리

### 예약/확인 필요 후보
- 히로카와: 장어, 예약/가격 확인 필수
- 인기 오코노미야키점: 미즈노/아지노야 등 웨이팅 고려
- USJ Express Pass: 닌텐도월드 우선순위 높으면 빠른 결정 권장

### 현실적인 베스트 조합
- Day 1 저녁: **아지노야 or 다루마 + 타코야키**
- Day 2: **USJ 내부 식사, 저녁은 우메다 가볍게**
- Day 3: **동쪽 교토는 간단히, 아라시야마에서 소바/카페**
- Day 4: **한큐/한신 식품관 + 551/디저트 선물**
