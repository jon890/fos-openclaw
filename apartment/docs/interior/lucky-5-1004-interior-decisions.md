### D-003. 샷시 판단 방식

- 상태: 방향 결정
- 현재 판단:
  - 전체 샷시 교체는 예산 부담이 커서 우선 범위에서 제외.
  - 거실 외창은 과거 한 번 교체된 것으로 보이나, 교체 시점이 오래됐고 완전 고급 사양은 아닌 것으로 추정.
  - 완전 구축 원래 샷시는 아니므로 무조건 교체로 단정하지 말고, 거실 확장 전제에서 실측 시 성능/상태를 보고 결정.
  - 방/주방 등 나머지 샷시는 상태 확인 후 유지·부분수리·필름/실리콘 보수 중심으로 판단.
  - 기존 거실 외창이 노후 이중창/일반 페어유리 수준이고 외풍·결로·기밀 문제가 있으면, 거실 확장 시 교체를 강하게 고려.
- 확인 필요:
  - 2015년 수리 때 거실 외창 샷시 교체 여부
  - 거실 외창 단창/이중창 여부
  - 거실 외창 페어유리 여부
  - 창틀 뒤틀림, 외풍, 물샘, 결로 흔적
  - 개폐/잠금 상태
  - 나머지 공간 샷시의 최소 보수 필요 여부

### D-004. 보일러 판단 방식

- 상태: 유지 쪽으로 기울어짐, 현장 확인 필요
- 현재 판단:
  - 매도인 설명상 보일러는 약 6년 전 교체.
  - 일반적인 교체주기를 10년 전후, 상태가 좋으면 10~15년까지도 보는 편이라 6년차면 즉시 교체 대상은 아님.
  - 난방/온수 상태가 정상이고 누수·소음·에러 이력이 없다면 이번 인테리어에서는 유지하는 쪽이 합리적.
  - 다만 인테리어 공사 중 배관/분배기 상태는 같이 확인하고, 제조년월·설치년월 사진으로 실제 연식을 검증.
- 확인 필요:
  - 제조년월/설치년월 사진
  - 난방/온수 상태
  - 소음, 에러코드 이력
  - 분배기/배관 누수 흔적
  - 배기통/연통 체결 상태와 안전 점검 여부

### D-005. 욕실 공사

- 상태: 임시 결정
- 내용:
  - 2015년쯤 한 번 수리했더라도 이미 약 10년 전이므로, 이번에 새로 하는 쪽에 무게.
  - 욕실은 입주 후 다시 공사하기 번거로우므로 공실일 때 처리하는 것이 유리.
- 선호:
  - 베이지 계열
  - 600각 또는 300×600 타일 검토
  - 환풍기 좋은 제품 고려

### D-006. 거실 베란다 확장 + 단열 강화

- 상태: 선호 확정, 상세 견적 필요

  - 확장 시 단열 강화 필요.
- 평면도 기반 추정:

- 비용 러프 추정:
  - 거실만 확장 + 거실 외창 샷시 교체: 약 700~1,000만원을 현실적인 기준으로 가정
  - 좋은 창호/단열 꼼꼼히: 1,000~1,300만원 이상 가능

- 견적 분리 요청 필요:
  1. 거실 발코니 확장만
  2. 거실 확장 + 단열 강화
  3. 거실 확장 + 거실 외창 샷시 교체
  4. 나머지 샷시 유지 시 필수 보수/실리콘/필름 등 최소 비용
  5. 전면 전체 샷시 교체 추가금은 참고용으로만 확인하거나, 견적 항목에서 제외 가능

  - 난방 배관 상태

  - 매트한 논슬립 베이지/그레이지 계열 샘플 확인

### D-008. 거실 확장

- 상태: 결정
- 내용:
  - 거실만 확장해서 넉넉하게 쓰는 방향으로 결정.
  - 안방 쪽 베란다와 기타 발코니는 비확장 유지.
  - 견적/관리사무소/구조 이슈 확인 전까지 공법과 세부 범위는 보류지만, 인테리어 기본 방향에서는 거실 확장을 포함.
  - 확장부 단열과 거실 외창 샷시 교체 필요 여부는 실측하면서 확인.
- 견적 비교 방식:
  - 기본 방향: 거실 확장 확정안.
  - 예산 비교: 확장 공사비, 단열/난방/샷시 추가금, 비확장 대안 비용을 분리해서 받기.
  - 최종 업체 선정 전에는 확장 시 하자 리스크와 비확장 대비 체감 효용을 함께 본다.
- 후속 확인:
  - 관리사무소 발코니 확장 가능 여부
  - 행위허가/신고 필요 여부
  - 난방 배관 연장 가능 여부
  - 거실 외창 샷시 교체 범위
  - 전체 샷시 미교체 시 나머지 샷시의 외풍/결로/누수 보수 가능 범위
  - 단열/결로 방지 상세 시공 방식

### D-008-1. 안방 쪽 베란다

- 상태: 결정
- 내용:
  - 안방 쪽 베란다는 비확장으로 유지.
  - 거실 확장에 예산과 하자 관리 포커스를 집중하고, 안방은 침실 기능과 수납 동선을 우선.
  - 다만 안방 베란다는 마감 개선을 진행: 바닥 타일 교체, 벽면 페인트, 끝쪽 수납장 제작 정도를 희망.
- 후속 확인:
  - 안방 베란다 창/문 개폐 상태
  - 결로·곰팡이·누수 흔적
  - 안방과 베란다 사이 기존 문/창 유지 또는 마감 보수 범위
  - 베란다 바닥 구배/배수 상태와 타일 미끄럼 여부
  - 벽면 페인트 가능 상태: 습기/곰팡이/들뜸 여부
  - 끝쪽 수납장 설치 시 깊이, 문 열림, 환기, 결로 간섭 여부

### D-009. 안방 가구 배치: 침실 중심 + 일자형 붙박이장

- 상태: 방향 결정
- 평면도 기반 추정:
  - 안방/룸#1 크기는 약 3420 × 4025mm로 추정.
  - 킹 또는 라지킹 침대 배치는 가능해 보임.
  - 붙박이장은 600~650mm 깊이를 고려하면 긴 벽 쪽 슬라이딩 도어형이 효율적.
- 현재 방향:
  - 안방은 `킹침대 + 붙박이장` 중심으로 구성.
  - 안방에는 책상을 두지 않는 방향.
  - 붙박이장은 기본 일자형으로 가는 방향. 방이 아주 넓지는 않아 ㄱ자/L자 수납은 답답해질 가능성이 큼.
- 확인 필요:
  - 침대 실제 사이즈
  - 출입문 스윙 방향
  - 창/커튼/콘센트 위치
  - 붙박이장 설치 가능 벽면 길이
  - L자 붙박이장 시 침대 양옆/발치 통로 폭

### D-010. 작업 공간: 작은방 서재 + 작은 모션데스크

- 상태: 결정 성향 강함

  - IKEA 1800 × 900 책상 보유.

- 현재 방향:
  - 기존 1800 × 900 책상은 매각/정리하고 더 작은 책상으로 전환하는 쪽.
  - 룸#2를 서재/작업방으로 사용.

  - 거실에 책상을 두는 안은 생활공간과 업무공간이 섞이고 시야가 지저분해질 수 있어 우선순위 낮음.
- 후속 확인:
  - 모니터 개수, PC 본체 위치, 스피커/마이크/조명 장비 여부
  - 작은방 책장/수납장 배치 필요 여부
  - 모션데스크 적정 폭: 1400 vs 1600

### D-011. 세탁실: 세탁기+건조기 직렬 설치 가능성

- 상태: 검토 후보
- 현재 보유/검토 제품:
  - LG 트롬 오브제컬렉션 세탁기 `FX24ENE`
  - LG 트롬 오브제컬렉션 건조기 `RD21ESE`
  - 세탁기+건조기 키트 직렬 설치 희망
  - 제품 치수 메모(공식 페이지 재확인 필요): 세탁기 약 W700 × H990 × D830mm, 건조기 약 W700 × H990 × D820mm 수준으로 보고 업체에 확인 요청.
- 평면도 기반 추정:
  - 세탁실은 주방 옆 좌하단 공간.
  - 세탁실 전체 폭 약 1,300mm, 깊이 약 1,100mm로 읽힘.
  - 창문은 `W865 × H870`, 하부 약 1,290mm 위치로 추정.
  - 도면상 공간만 보면 700mm급 세탁기/건조기 직렬 설치 가능성은 있음.
- 현재 판단:
  - 럭키 24평 평면도 참고 기준상 병렬 설치는 어려울 가능성이 큼.
  - 도면상 주방/세탁 관련 좌측 폭이 약 1,200mm로 읽혀 700mm급 세탁기+건조기 병렬은 여유폭/문열림/급배수까지 고려하면 비현실적.
  - 직렬 설치를 1순위로 보고, 실제 결정은 높이·문폭·배수/급수/콘센트 위치 확인 필요.
  - 직렬 설치가 불가능하면 건조기만 베란다 배치하는 대안을 검토.
  - 단, 베란다 배수관이 우수관이면 세탁 배수/건조기 배수호스를 연결하지 않는 방향이 안전하며, 건조기는 물통 비움 방식 또는 관리사무소 확인 후 판단.
  - 직렬 설치 시 창문을 일부 가리거나, 창 개폐/환기/블라인드와 간섭할 수 있음.
  - 제품 공식 치수와 LG 설치 기준 확인 필요.
- 현장 실측 필요:
  - 세탁실 문틀 내측 폭
  - 세탁실 내부 폭/깊이/천장고
  - 급수꼭지 돌출 위치와 높이
  - 배수구 위치
  - 콘센트 위치와 단독 회로 여부
  - 창문 하단 높이와 개폐 방향
  - 설치 후 세탁기/건조기 도어 개폐 공간
  - 업체/LG 설치 기준상 직렬 키트 적용 가능 여부

### D-012. 안방 앞/화장실 입구 벽: 화장대 + 드라이 공간

- 상태: 진행 방향 결정, 실측 확인 필요
- 내용:
  - 안방에서 나와 화장실 입구 쪽 벽면에 작은 화장대를 설치하고 싶음.
  - 머리 말리는 공간으로도 쓰고 싶으므로 드라이기 사용 가능한 콘센트 확보가 필요.
- 1차 평가:
  - 동선상 장점이 큼. 안방 → 화장대 → 화장실/외출 동선이 자연스럽고, 안방 안에 화장대를 넣지 않아 침실을 더 깔끔하게 유지할 수 있음.
  - 20평대에서는 독립 파우더존 느낌을 만들 수 있어 만족도가 높을 가능성이 있음.
  - 다만 화장실 입구 주변은 습기, 통행폭, 문 간섭, 조명 그림자, 콘센트 위치를 반드시 확인해야 함.
- 설계 권장:
  - 폭이 허용되면 얕은 깊이의 벽부형/하부장형 화장대 검토.
  - 상부 거울장 또는 간접조명 거울을 쓰면 공간 효율이 좋음.
  - 드라이기, 고데기, 칫솔살균기/충전기 등을 고려해 콘센트는 최소 2구 이상 권장.
  - 가능하면 화장대 상판 위쪽 또는 측면에 콘센트를 배치하고, 바닥 가까운 콘센트만 두는 것은 피함.
- 전기/안전 확인:
  - 드라이기 사용 전력을 고려해 회로 여유 확인 필요.
  - 욕실 물튀김 직접 영향권이면 방수형 콘센트/위치 조정 검토.
  - 조명 스위치와 콘센트 위치가 문틀/수납장/거울과 간섭하지 않게 전기 도면에 반영.
- 현장 실측 필요:
  - 화장실 문 여닫이 방향과 문 열림 반경
  - 안방 문/복도 동선 간섭
  - 설치 가능 벽면 폭과 통행 잔여 폭
  - 기존 콘센트/스위치 위치
  - 욕실 앞 습기/환기 상태

### D-013. 시스템에어컨

- 상태: 결정
- 내용:
  - 시스템에어컨은 설치하지 않는 방향으로 결정.
  - 예산 부담, 천장/배관 공사 부담, 구축 20평대에서의 비용 대비 효용을 고려해 이번 공사 범위에서 제외.
- 견적 요청 반영:
  - 시스템에어컨 항목은 기본 견적/옵션 견적에서 제외.
  - 냉방은 기존 또는 별도 벽걸이/스탠드 에어컨 운용을 전제로 판단.
- 후속 확인:
  - 기존 에어컨 배관/실외기 위치 사용 가능 여부
  - 거실 확장 후 냉방 효율에 문제가 없는지 업체 의견 확인

### D-014. 중문

- 상태: 설치 방향, 타입 추후 결정
- 내용:
  - 중문은 설치하는 방향.
  - 사용자는 여닫이 선호 의견이 있는 배우자와 추가 의논 후 결정 예정.
  - 슬라이딩과 여닫이 중 선택은 평면/현관 폭/신발장 간섭 확인 후 결정.
- 현재 추천:
  - 24평 구축 평면에서는 여닫이보다 슬라이딩 계열을 우선 추천.
  - 이유: 여닫이는 문 열림 반경이 필요해 현관/복도/신발장 간섭이 생기기 쉽고, 공간이 더 답답해질 수 있음.
  - 단, 슬라이딩은 옆으로 빠질 벽면 폭이 필요하므로 현장 실측이 필수.
- 후보:
  - 3연동 슬라이딩: 가장 무난. 단열/소음 차단과 공간 효율 균형이 좋음.
  - 원슬라이딩/초슬림 슬라이딩: 디자인은 깔끔하나 개구폭/벽면 조건 확인 필요.
  - 여닫이/스윙도어: 공간이 충분하거나 디자인상 꼭 필요할 때만 검토.
- 현장 확인:
  - 현관에서 복도로 이어지는 폭
  - 신발장/현관장 간섭
  - 문 열림 반경
  - 슬라이딩 레일 설치 가능 폭
  - 로봇청소기 문턱/레일 간섭 여부
