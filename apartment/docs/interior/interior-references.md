## 목적

아침 인테리어 레퍼런스 다이제스트에서 발견한 후보 중, 나중에 업체 미팅/견적 요청/스타일 결정에 다시 볼 만한 링크를 누적한다.

- 결정 확정 문서는 `lucky-5-1004-interior-decisions.md`
- 이 문서는 레퍼런스/업체/아이디어 후보 보관용
- 결정으로 승격하려면 사용자 확인 후 의사결정 노트에 `D-00X`로 반영

## 현재 핵심 기준

- 대상: 구리 럭키아파트 5동 1004호
- 예산: 기본 3,000만원 전후, 필요 시 증액 가능
- 스타일: 화이트/베이지 + 우드 포인트
- 우선 관심사:
  - 거실 베란다 확장
  - 단열 강화
  - 거실 샷시 교체 여부
  - 욕실/주방/조명/문/몰딩/전기
  - 안방 붙박이장
  - 작은방 작업실/모션데스크 배치
  - 공용부 600각 타일 + 방 마루 조합 가능성
