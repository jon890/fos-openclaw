# 구리 럭키아파트 5동 1004호 인테리어 의사결정 요약

- 기준일: 2026-05-13
- 원본 의사결정 노트: `lucky-5-1004-interior-decisions.md`
- 목적: 지금까지 정해진 방향을 견적 상담/검토용으로 한눈에 보기

## 1. 기본 방향

- 대상: 구리 럭키아파트 5동 1004호
- 잔금일: 2026-09-09
- 입주: 인테리어 완료 후 입주 예정
- 1차 예산: 약 3,000만원 전후
- 스타일: 밝고 따뜻한 화이트/베이지 톤 + 과하지 않은 우드 포인트
- 전략: 기본 공사와 옵션 공사를 분리해서 견적 비교
- 예산 관점: 잘 수리해서 실거주하다가 향후 갈아타기까지 고려

## 2. 현재 결정된 큰 범위

### 기본 포함 후보

- 도배
- 바닥재
- 욕실
- 싱크대
- 조명
- 문/몰딩
- 전기, 콘센트, 스위치

### 옵션/상태 확인 후 판단

- 거실 외창 샷시 교체
- 보일러 유지/점검
- 중문
- 붙박이장/제작가구
- 시스템에어컨: 설치하지 않음으로 결정
- 안방 앞/화장실 입구 화장대

## 3. 거실 확장

- 상태: 결정
- 방향: 거실 베란다는 확장해서 넓게 쓰는 쪽
- 이유:
  - 20평대에서 거실 체감 면적 증가 효과가 큼
  - 레퍼런스 방향과도 잘 맞음
  - 실거주 만족도에 직접 영향
- 단, 최종 전에는 관리사무소/업체 확인 필요:
  - 발코니 확장 가능 여부
  - 행위허가/신고 필요 여부
  - 난방 배관 연장 가능 여부
  - 단열/결로 방지 시공 방식
  - 확장부 바닥 단차 처리

## 4. 샷시

- 상태: 방향 결정
- 방향: 전체 샷시 교체는 우선 제외, 거실 외창 샷시 교체 중심으로 검토
- 이유:
  - 전체 샷시 교체는 예산 부담이 큼
  - 거실 확장 시 가장 중요한 부위는 거실 외창
  - 방/주방 쪽은 상태가 나쁘지 않다면 유지 또는 부분 보수로 충분할 수 있음
- 견적 요청 방향:
  - 거실 확장 + 거실 외창 샷시 교체 기준
  - 나머지 샷시는 유지 전제
  - 필요 시 실리콘, 레일, 잠금장치, 필름 등 부분 보수만 확인
- 현장 확인:
  - 거실 외창 단창/이중창 여부
  - 페어유리/로이유리 여부
  - 창틀 뒤틀림, 외풍, 물샘, 결로 흔적
  - 개폐/잠금 상태

## 5. 보일러

- 상태: 유지 쪽으로 기울어짐
- 현재 정보: 약 6년 전 교체했다고 들음
- 판단:
  - 일반적인 보일러 교체주기는 10년 전후
  - 상태가 좋으면 10~15년까지도 사용 가능
  - 6년차가 맞다면 이번 인테리어 때 선제 교체할 필요는 낮음
- 방향:
  - 정상 작동이면 유지
  - 공사 전 설비기사 또는 업체를 통해 점검
- 확인할 것:
  - 제조년월/설치년월 사진
  - 모델명/용량
  - 온수 온도 안정성
  - 난방 순환 상태
  - 소음, 에러코드, 누수 여부
  - 분배기/배관 누수·녹·부식 흔적
  - 연통/배기통 체결 상태

## 6. 욕실

- 상태: 임시 결정
- 방향: 이번에 새로 공사하는 쪽에 무게
- 이유:
  - 2015년쯤 수리했더라도 이미 약 10년 전
  - 욕실은 입주 후 다시 공사하기 번거로움
  - 공실일 때 방수/배관/타일을 같이 보는 것이 유리
- 선호:
  - 베이지 계열
  - 600각 또는 300×600 타일 검토
  - 환풍기 좋은 제품 고려
- 견적서 확인:
  - 철거 범위
  - 방수 횟수와 범위
  - 배관/설비 교체 범위
  - 환풍기 제품명 또는 성능

## 7. 바닥재

- 상태: 보류
- 현재 판단:
  - 600×600 타일은 고급감과 확장감이 좋음
  - 다만 방까지 전체 적용은 차가움, 딱딱함, 하자 리스크, 매도 호불호 때문에 신중
- 추천 방향:
  - 거실 + 주방 + 복도 + 현관/확장부: 600각 타일 검토
  - 방: 강마루/마루 계열 추천
- 확인할 것:
  - 바닥 수평 상태
  - 난방 배관 상태
  - 타일 시공 시 셀프레벨링/몰탈 보정 필요 여부
  - 마루와 타일 경계 단차

## 8. 안방

- 상태: 결정 성향 강함
- 방향: 침실 중심 + 붙박이장 우선
- 구성:
  - 킹 또는 라지킹 침대
  - 긴 벽 쪽 슬라이딩 붙박이장 우선 검토
  - 안방에는 책상을 두지 않는 방향
- 확인할 것:
  - 침대 실제 사이즈
  - 출입문 스윙 방향
  - 창/커튼/콘센트 위치
  - 붙박이장 설치 가능 벽면 길이
  - L자 붙박이장 시 통로 폭

## 9. 작업 공간

- 상태: 결정 성향 강함
- 방향: 작은방을 서재/작업방으로 사용
- 현재 방향:
  - 기존 IKEA 1800×900 책상은 정리/매각 검토
  - 1400~1600 × 700~800급 모션데스크 우선 검토
  - 거실 책상 배치는 우선순위 낮음
- 확인할 것:
  - 모니터 개수
  - PC 본체 위치
  - 책장/수납장 필요 여부
  - 콘센트/랜선 위치
  - 의자 뒤 공간

## 10. 세탁실

- 상태: 검토 후보
- 희망:
  - LG 트롬 오브제컬렉션 세탁기 `FX24ENE`
  - LG 트롬 오브제컬렉션 건조기 `RD21ESE`
  - 세탁기+건조기 직렬 설치
- 현재 판단:
  - 도면상 가능성은 있어 보이나 실측 필수
  - 창문, 급수/배수, 콘센트, 문폭 간섭 확인 필요
- 확인할 것:
  - 문틀 내측 폭
  - 내부 폭/깊이/천장고
  - 창문 하단 높이와 개폐 방향
  - 급수꼭지 돌출 위치
  - 배수구 위치
  - 콘센트 위치와 단독 회로 여부

## 11. 안방 앞/화장실 입구 화장대

- 상태: 검토 후보, 선호 있음
- 방향:
  - 안방에서 나와 화장실 입구 쪽 벽면에 작은 화장대 설치 검토
  - 머리 말리는 공간으로도 사용
- 장점:
  - 안방을 더 깔끔하게 유지 가능
  - 화장실/외출 동선과 잘 맞음
  - 20평대에서 독립 파우더존 느낌을 줄 수 있음
- 주의:
  - 습기
  - 통행폭
  - 문 간섭
  - 조명 그림자
  - 콘센트 위치
- 요청 방향:
  - 얕은 깊이의 벽부형/하부장형 화장대
  - 거울 또는 간접조명 거울
  - 드라이기용 콘센트 최소 2구 이상

## 12. 시스템에어컨

- 상태: 결정
- 방향: 설치하지 않음
- 이유:
  - 예산 부담이 큼
  - 천장/배관 공사 부담이 있음
  - 구축 20평대에서 비용 대비 효용이 애매할 수 있음
- 대신 확인할 것:
  - 기존 에어컨 배관/실외기 위치 사용 가능 여부
  - 거실 확장 후 냉방 효율 문제 여부

## 13. 업체 견적 요청 시 기본 문장

> 구축 20평대 아파트이고, 화이트/베이지+우드톤으로 밝고 따뜻하게 하고 싶습니다. 예산은 기본 3천 전후이고, 거실 베란다 확장과 단열 강화는 우선 검토하고 싶습니다. 샷시는 전체 교체보다는 거실 외창 교체 중심으로 보고, 나머지는 상태 확인 후 유지/부분보수로 판단하고 싶습니다. 보일러는 6년 전 교체된 것으로 들어서 정상 작동하면 유지하고, 분배기/배관/누수/난방 순환 상태만 점검하고 싶습니다. 중문/붙박이장/안방 앞 화장대는 현장 상태와 견적 보고 옵션으로 판단하고 싶습니다. 기본 견적과 옵션 견적을 나눠서 받을 수 있을까요?

- 기본 3,000만원안
- 거실 확장 + 단열 추가안
- 거실 외창 샷시 교체 추가안
- 나머지 샷시 유지 시 필수 보수 비용
- 비확장/폴딩도어 대안안
- 보일러 점검 및 필요 시 교체 추가안
- 중문 추가안
- 붙박이장/제작가구 추가안
- 안방 앞/화장실 입구 화장대 + 거울/조명 + 콘센트 신설 추가안
- 공용부 600각 타일 + 방 마루 조합안

3. 보일러 6년 전 교체가 실제 명판으로 확인되는가?
